## 青蛙跳扩展

一只青蛙一次可以跳上1级台阶，也可以跳上2级。。。它也可以跳上n级。此时青蛙跳上一个n级的台阶总共有多少种跳法？我们用数学归纳法可以证明f(n)=2^(n-1)
原因：
跟变态青蛙跳一样：
f(n) = f(n-1) + f(n-2) +... + f(1) + 1
之所以最后还要加上1， 是因为青蛙可以选择直接跳n级台阶到目的地，而这个解法没有包含在前面的几种跳法里。
归纳证明：

```
f(1) = 2^0 = 1

f(2) = 2^1 = 2

f(n) = f(n-1) + f(n-2) +... + f(1) + 1
     = 1 + f(1) + ... + f(n-2) + f(n-1)
     = 1 + 2^0 + 2^1 + 2^2 + ... + 2^(n-2)
     = 2^0 + 2^0 + 2^1 + 2^2 + ... + 2^(n-2)
     = 2^1 + 2^1  + 2^2 + ... + 2^(n-2)
     = ...
     = 2^(n-2) + 2^(n-2)
     = 2^(n-1)
```
